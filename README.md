===================
CNV ANALYZER
===================

Python script to cross copy number window values versus gene bed coordinates
   
Licensing
=========

cnv_analyzer script are licensed under GPL.

Installation
============

Download the script and be sure Bedtools is installed in your system. Then run it.

=============
